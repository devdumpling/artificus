1. Clone the repo
2. Create a new branch with your name and date of the addition (e.g. `git checkout -b tim-2022-9-14`)
3. Add your notes to the `pages/player-notes` directory. You can use Markdown or HTML to format.<sup>1</sup>
4. Commit your changes and push them to your branch
5. Create a pull request to merge your branch into `main`
6. Request review from the DM `@devdumpling` or DM Dev on discord. 
7. Once approved, merge (or ask Dev to do it if you're confused).

Thanks for adding notes to the docs site!
